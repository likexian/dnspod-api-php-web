# DNSPod API PHP Web 示例

## 功能说明
用PHP实现了一个 DNSPod API 的 Web 示例，已完成对域名和记录的基本操作，可直接使用。

功能包括：
- 用户登录
- 域名列表
- 域名暂停/启用
- 域名添加
- 域名删除
- 记录列表
- 记录暂停/启用
- 记录添加
- 记录修改
- 记录删除
- D令牌验证

## 环境要求
- PHP5.x
- SESSION
- JSON
- CURL

## 安装说明
直接放到网站的任何目录，然后在浏览器打开即可查看示例。
